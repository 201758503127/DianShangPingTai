更新用户信息

在线测试 在线浏览

请求方式：POST

请求地址：http://localhost:8080/actionmall/mgr/user/updateuser.do

Body参数名 类型 必需 描述 示例 e.g.
id string 是 用户id 1
name string 是 姓名 小明
account string 是 用户名 admin
age string 是 年龄 18
phone string 是 电话 17844545452
email string 是 邮箱 546415155@qq.com

详细说明：

该接口用于管理员保存修改完成的用户信息。
